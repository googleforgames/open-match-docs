{{< /blocks/cover >}} {{% blocks/lead color="secondary" %}} What is Open Match?

Open Match is a flexible match making system built to scale with your game. {{% /blocks/lead %}} {{< blocks/section color="blue" >}} {{% blocks/feature icon="fa-code" title="Flexible" %}} Write code, not config, to determine how matches should be made. {{% /blocks/feature %}} {{% blocks/feature icon="fa-tachometer-alt" title="Scale" %}} Built to scale up and down quickly. For games big and small. {{% /blocks/feature %}} {{% blocks/feature icon="fa-chart-area" title="Measurable" %}} Measure quality and latency of matches. Easily run experiments to find the right balance. {{% /blocks/feature %}} {{< /blocks/section >}} {{< blocks/section color="primary" >}}
